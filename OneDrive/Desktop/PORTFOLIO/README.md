## How to Use

### Local Development
1. Extract or navigate to the portfolio folder
2. Open `index.html` in your web browser
3. No server required for basic viewing

### File Hosting Options

**Option 1: GitHub Pages (Recommended)**
1. Create a GitHub repository
2. Upload these files to the repository
3. Enable GitHub Pages in repository settings
4. Access at: `https://yourusername.github.io/repository-name`

**Option 2: Netlify**
1. Go to netlify.com
2. Drag and drop the portfolio folder
3. Automatic deployment with custom domain option

**Option 3: Vercel**
1. Go to vercel.com
2. Import the project
3. Automatic CI/CD deployment

**Option 4: Web Server**
1. Upload files to any web hosting provider
2. Access via your domain

## Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --neon-green: #00ff88;    /* Change neon green */
    --neon-blue: #00d9ff;     /* Change neon blue */
    --primary-bg: #0a0e27;    /* Change background */
    --text-primary: #ffffff;  /* Change text color */
}
```

### Update Content
Edit `index.html` to:
- Change headline and sub-headline
- Update bio and descriptions
- Modify achievement numbers
- Add/remove courses in mentorship section
- Update service descriptions
- Change contact email

### Add Social Links
In the Contact section, update the social icon links:
```html
<a href="your-linkedin-url" class="social-icon" title="LinkedIn">in</a>
<a href="your-twitter-url" class="social-icon" title="Twitter">tw</a>
<a href="your-github-url" class="social-icon" title="GitHub">gh</a>
```
